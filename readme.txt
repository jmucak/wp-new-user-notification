Customize new user notification message

== Description ==

You can customize new user notification message.

== Installation ==

Upload 'wp-new-user-notification' to the '/wp-content/plugins/' directory

Activate the plugin through the 'Plugins' menu in WordPress

= How this plugin works =

1. Open Tools -> WP New User notification
2. Type new message
